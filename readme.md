# Penguin Fashion

Welcome to **Penguin Fashion**. This is a simple and beginner-friendly project that focuses on creating a visually appealing and responsive landing page for a fashion store to showcase its offerings and featured products. [Click here](https://sajjad1007.github.io/Penguin-Fashion/) to visit the website.

## Technologies Used

1. **HTML5:** For structuring the website.
2. **Tailwind CSS:** For styling the layout and responsive design.

## How To Use

1. Clone the repository:
   ```bash
   git clone https://github.com/Sajjad1007/Penguin-Fashion.git
   ```
2. Open the project folder and launch `index.html` in your browser.
